# yolov5_infer

本代码是对yolov5-7.0代码的封装，对于yolov5其他版本的代码可能需要更改一些东西。

代码对一些模块进行封装，方便使用。

代码参数调整可在main_detect.py里面修改

代码中的detect.py为官方的代码，可供调试。

代码部署

```python
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

部署好后，运行代码

```python
python main.py
```
